Configuration Guide

Die zentrale Konfigurationsdatei ist die Datei jobmatch.conf im Verzeichnis jobmatch/jobmatch. Nachfolgend sind die wichtigsten Konfiguartionsmoeglichkeiten erklaert.

Datenbankkonfiguration


1. Tabellen erstellen

Die Tabellen der Datenbank koennen mit der Datei create_tables.sql im Verzeichnis /doc generiert werden. Dazu muss man eine neue Datenbank erstellen und beim einloggen create_tables.sql als Parameter mitgeben.

Mit einer Mysql-Datenbank wuerde dieser Aufruf folgendermassen aussehen:

mysql -h "Datenbank-Server" -u "Kontoname" -pMyPassword "Datenbankname" < create_tables.sql

2. Datenbank ans System anbinden

Nun muss in der Datei jobmatch.conf im Abschnitt "Database Manager Configuration" folgende Aenderungen gemacht werden:
Kontonamen und Passwort angeben:
DatabaseManager.DB.Kontaktboerse4.Connection.User = "Kontoname"
DatabaseManager.DB.Kontaktboerse4.Connection.Password = "Passwort"

Datenbanktreiber und URL des Datenbankservers angeben. Dies sieht mit einer Mysql-Datenbank unter der Adresse leeta.unibe.ch:3306 folgendermassen aus:
DatabaseManager.DB.Kontaktboerse4.JdbcDriver = org.gjt.mm.mysql.Driver
DatabaseManager.DB.Kontaktboerse4.Connection.Url = jdbc:mysql://leeta.unibe.ch:3306/Kontaktboerse4

Weitere Konfigurationsmoeglichkeiten

Im Abschnitt "Jobmatch Settings" koennen folgende Werte gesetzt werden:

  • Jobmatch.RootPassphrase = "Passwort"
    setzt das Passwort fuer den Administrator

  • Jobmatch.Mailer.SMTPServer = "localhost"
    setzt den SMTPServer fuer das Mailing

  • Jobmatch.JobScript.Directory = "../Skriptverzeichnis"
    setzt das Script-Verzeichnis
