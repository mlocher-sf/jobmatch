Administrator User Guide

Start des Systems

Jobmatch wird mit dem start-Skript im Unterverzeichnis jobmatch/output gestartet und steht anschliessend auf Port 9004 zur Verfuegung

Stop des Systems

Jobmatch wird beendet, indem der zugehoerige Java Prozess terminiert wird. (zB mittels kill)

Laufzeiteinstellungen

Alle Einstellungen die der/die Administrator zur Laufzeit ausfuehren kann, sind unter der URL #JOBMATCH#/admin/ControlCenter.po erreichbar.
Die Verwaltung der Administratoren- und Firmen Konten ist selbsterklaehrend.

JPython Scripting

Jobmatch verfuegt ueber eine eingebaute Skripting Engine.
Dokumentation zu JPython

Der Benutzer kann die Funktionen von Jobmatch in seinen Skripts benutzten.

Interpreter

Mit dem Interpreter werden zeilenweise Befehle versandt.

Skripts ausführen

Mit dem JobScript Runner werden ganze Skript Dateien abgearbeitet.
Der Benutzer kann selbst neue erstellen, indem er Python Skripts in das bei der Konfiguration angebenen Verzeichnis kopiert. Die Endung auf .pz ist zwingend.
